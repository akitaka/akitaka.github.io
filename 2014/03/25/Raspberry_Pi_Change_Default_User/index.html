下面就来改改默认的用户名和密码。

先介绍下最简单的改密码方法:

  1. 登录后输入sudo raspi-config
  2. 选择2 Change User Password。
  3. 输入你的密码

    完成～这样就已经修改了默认的密码。当然这样我们的用户名依然是默认的pi，也只有在默认pi的用户名下可以如此修改密码。
